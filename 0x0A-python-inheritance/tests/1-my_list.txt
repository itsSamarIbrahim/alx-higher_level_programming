=============================
The ``1-my_list`` module
=============================

Using ``MyList``
---------------------

Import the class:

    >>> MyList = __import__('1-my_list').MyList

Now test it:

>>> my_list = MyList([3, 1, 2])
>>> my_list.print_sorted()
[1, 2, 3]

>>> empty_list = MyList()
>>> empty_list.print_sorted()
[]

>>> another_list = MyList([10, 5, 8, 3, 6])
>>> another_list.print_sorted()
[3, 5, 6, 8, 10]

>>> my_list.append(4)
>>> my_list.print_sorted()
[1, 2, 3, 4]

>>> my_list.extend([0, 9])
>>> my_list.print_sorted()
[0, 1, 2, 3, 4, 9]

>>> my_list.remove(2)
>>> my_list.print_sorted()
[0, 1, 3, 4, 9]

>>> my_list.clear()
>>> my_list.print_sorted()
[]
